# JOAF — Design System & UI Guidelines
**জুলাই অনলাইন অ্যাক্টিভিস্ট ফোরাম**
Version: 2.0 | July 2026

## 1. Brand Identity

**Name:** JOAF / জুলাই অনলাইন অ্যাক্টিভিস্ট ফোরাম  
**Mission:** Democracy, civic rights, and public interest publishing for Bangladesh  
**Tone:** Trustworthy, civic, Bengali-first, youth-oriented, urgent when needed  
**Visual Identity:** Dark backgrounds, red accent (resistance/urgency), Bengali typography

## 3. Typography

### Admin / Web

| Usage | Font | Weight | Size |
|---|---|---|---|
| Body (Bengali) | Noto Sans Bengali | 400–900 | 13–16px |
| UI (Latin) | Segoe UI / system-ui | 400–800 | 11–16px |
| Sidebar logo | Segoe UI | 900 | 14px |
| Admin headings | Noto Sans Bengali | 800 | 18–24px |
| Data labels | Segoe UI | 700 | 10px (uppercase, letter-spacing 0.6px) |
| Code/IDs | monospace | 400 | 11px |

### Content Cards (server-side generated)

| Usage | Font | Weight | Notes |
|---|---|---|---|
| Hook text | NotoSerifBengali | Bold | Bundled TTF in `fonts/` |
| Body text | NotoSerifBengali | Regular | Bundled TTF |
| Badge label | HindSiliguri | Bold | Bundled TTF |
| Caption | HindSiliguri | SemiBold | Bundled TTF |
| JOAF watermark | HindSiliguri | Regular | Bottom-right corner |

**Critical:** Bengali fonts MUST be bundled as TTF files in the function's `fonts/` directory. Appwrite Alpine Linux has no system Bengali fonts. Never rely on Google Fonts CDN in server-side rendering.

## 7. Accessibility & Localization

- **Primary language:** Bengali (বাংলা)
- **Secondary language:** English (UI labels, code, IDs)
- **Font loading:** Google Fonts (Noto Sans Bengali 400–900) for web UI
- **Bundled fonts:** NotoSerifBengali, HindSiliguri for server-side generation
- **Text direction:** LTR (Bengali is LTR)
- **Charset:** UTF-8 throughout
- **HTML lang:** `lang="bn"` on Bengali pages, `lang="en"` on Studio
- **Mobile:** All admin pages responsive with sidebar collapse at 768px

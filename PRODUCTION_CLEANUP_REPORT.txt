IMPORTANT — YOU MUST SET THESE ENV VARS IN NETLIFY:
  APPWRITE_API_KEY     → your Appwrite server API key
  ADMIN_SECRET_KEY     → your admin password
  VAPID_PUBLIC_KEY     → web push public key
  VAPID_PRIVATE_KEY    → web push private key

  Firebase env vars (FIREBASE_PROJECT_ID, FIREBASE_PRIVATE_KEY,
  FIREBASE_CLIENT_EMAIL) can be REMOVED from Netlify after migration.
